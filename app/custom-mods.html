The full modding toolchain, in your browser. Install community mods, or create / pack / build your own - the same tools the desktop apps run, over your already-imported game.

Import a mod

A mod is a diff: it carries only the author's changes, none of the game's content, so your imported base game is required. Imported mods appear in the in-game Mods menu, where you can enable, disable or uninstall them.

Create an editable project

Recover an editable RPG Maker MV project from a game's www - decrypted data, named assets and baked dialogue, ready to open in the editor. Downloads as a .zip.

Runs entirely in your browser. Image compositing (parallax map backgrounds) needs a native canvas and is skipped here - data, assets and text are recovered fully.

Pack an edited project

Re-encode an edited project back into a TCOAAL-shaped www (re-hashed names, re-encrypted assets, un-baked dialogue). Downloads as a .zip you can re-import on the loader or run with node server.js.

Build a shareable mod

Diff your edited project against one or more base game versions to produce a copyright-safe .tcoaalmod - it ships only your changes, never the game's content. Players supply their own game.
