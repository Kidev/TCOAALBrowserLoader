Create an editable project

Turn your game into an editable RPG Maker MV project you can open in the editor and start changing. Downloads as a .zip.

Build a shareable mod

Package your edits into a mod file you can share. It contains only your changes and none of the game's content. That makes it legal and DMCA-safe to distribute: no copyrighted game files are ever included.

Building packs your project to verify your edits, then compares it against the selected game versions to create a mod file carrying only your changes. This can take a while.
