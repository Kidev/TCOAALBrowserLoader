TCOAAL Mod Creator

Create a project from your game, edit it in RPG Maker MV, test it in the browser, then share it as a copyright-safe .tcoaalmod diff.
Create project from a game
Decrypt a game folder into an editable RPG Maker MV project.
Test in browser (dev loop)
Run the project in true browser mode and open it in your browser.
Pack project (re-encode for the game)
Re-encode the edited project back into a playable www folder.
Share (build .tcoaalmod)
Diff the edited project against one or more base game versions to produce a copyright-safe, shareable .tcoaalmod.
Download game versions from Steam
Detect your Steam install and download a specific TCOAAL version's depot. Each downloaded version is archived permanently for this app, safe from later Steam updates, and can be added as a base in the Build mod tab.
